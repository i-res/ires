Welcome to iRES.

iRES is a loader that allows you to call other libraries js here and there. in an easy and dynamic way.





i-res.github.io/ires

The forms by default and mandatory to invoke a library js is as follows:
*The loader is used by calling the name of the library, like a variable that enters a function.
*Then call the version that you want to load.
The default loader calls the minified version, if any.

<html>
<head>
<meta http-equiv="content-type" content="text/html; charset=UTF-8">
<title>My WebSite</title>
<script type="text/javascript" src="http://i-res.github.io/ires/ires.js"></script>
<script type="text/javascript">ires.load("jquery", "1.4.3");</script>
</head>
<body>
</body>
</html>

After calling the loader with the correct options, will be added a <script> tag corresponding to the library.

<script type="text/javascript" src="http://i-res.github.io/ires/ires.js"></script>
<script type="text/javascript">ires.load("jquery", "1.4.3");</script>


<script src="http://i-res.github.io/ires/libs/jquery/jquery-1.4.3.min.js" type="text/javascript"></script>

You can summon charger with these optional values.
*{minify:true} or {minify:false} load the minified version if it is marked as true.
*"SSV2" or "SSV3" load the resource from an alternate server. Some resources have two defined servers, others only one.
*"SSV2" is used to load the resource from Google.
*"SSV3" is used to load the resource from the website of the owner of the resource.

<script type="text/javascript" src="http://i-res.github.io/ires/ires.js"></script>
<script type="text/javascript">ires.load("jquery", "1.4.3", {minify:false}, "SSV3");</script>


<script src="http://code.jquery.com/jquery-1.4.3.js" type="text/javascript"></script>
